Check-in new revision
=====================

* update CHANGELOG if necessary
* run test suite
* run doctests


Release a new version
=====================

Tests
-----

* run the test suite 
  $ python runtest.py

* make sure tests work with all supported python versions, e.g.
  $ python runtest.py -P python2.4

* run doctests on the documentation
  $ cd doc
  $ make doctest

* check sources with pychecker
  $ python runtest.py --pychecker

Checks
------
  
* check the meta data specified in pympler/metadata.py
* check CHANGELOG file for missing entries
* check NOTICE file for missing contributors
* check the to-be-released module-distribution
  $ python setup.py sdist

Release
-------
  
* release a new version to the python cheeseshop via
  $ python setup.py sdist --formats=gztar,zip upload
* update the documentation at http://packages.python.org/pympler/
